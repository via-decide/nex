# Nex electronics research domain pack

Nex discovers official electronics sources, preserves artifacts, extracts source-grounded candidates, records conflicts, supports human review, and exports deterministic research packs. Nex does **not** confirm board components, authorize hardware access, flash firmware, erase storage, operate JTAG/relays/power supplies, or set board verification status; via-decide/electronics consumes reviewed packs and performs deterministic validation.
